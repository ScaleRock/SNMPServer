**SNMP Server**, as the name suggests, is intended to be a simple implementation of the SNMP protocol. The aim of the project is to enable easy and platform-independent implementation of SNMP.

Ultimately, snmp v1, v2c, and v3 will be supported. We accept the following as the source of the standard for specific versions:
* v1 :
  * rfc1157
* v2c:
  * rfc1901
  * rfc1905
  * rfc1906
  * rfc1907
* v3:
  * rfc3411
  * rfc3412
  * rfc3413
  * rfc3414
  * rfc3415
  * rfc3416
